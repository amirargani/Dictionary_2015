## 🎵 MP3 Playback Windows Forms Project

This project is a simple Windows Forms application developed in **C# (.NET Framework)** that demonstrates continuous MP3 playback.

---

## 🧩 Project Overview

The goal of this project was to design a Windows Form that:

- 🎶 Plays an MP3 file continuously in the background
- ✨ Opens and closes with a visual effect (e.g., fade-in/fade-out or slide animation)

It serves as a basic example of integrating media playback and UI transitions in a desktop application.

---

## 🛠️ Technologies Used

- **Language:** C#
- **Framework:** .NET Framework / Windows Forms
- **Media Playback:** `Windows Media Player`
- **UI Effects:** Animation using timers and form properties
